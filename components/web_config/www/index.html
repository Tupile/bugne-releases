Saved networks
Passwords are never shown. Add as many networks as you like; the device uses the strongest in range and roams between them. Tap a network above to fill it in, or type the name for hidden networks. Leave a password blank to keep the one already saved.

Advanced: upload a file
Upload a firmware file (bugne.bin) from your computer to update over Wi-Fi. Keep the device powered; it reboots into the new version when done.
